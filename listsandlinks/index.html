Unit 5: Assignment

This assignment's goal is to prove to the teacher wheather we know how to create a webpage and run it in the computer, and as well as turing it in to the public page. We learn how to create an ordered list, unordered lists, and hwo to add in links into our webpage. Mr.Keiffer doesn't tell us exactly what to do but he lets us research it on our own.

  • Create a new folder called "listsandlinks" in your repository folder
  • Inside listsandlinks create a file called "index.html" and a file called "main.css"
  • In index.html create an ordered list of step by step instructions to create a new webpage on the production server.
  • Also create an unordered list of 5 or more links to websites with useful webdesign related information. In the nav section, create an unordered list of links with a link back to your main index page.
  • In your main index.html file in public_html, add an unordered list of links to the nav section with a link to "listsandlinks".
  • Add an article with a heading and a paragraph description of this assignment. Also add an unordered list that contains this list of requirements.
  • Turn in your work.

Step by Step Instructions to Create a New Webpage on the Production Server

  1. Clone your repository
  2. Go into your folder and create a new file (gedit 'filename.html'
  3. Edit the file and save it
  4. In Terminal, create a new file by using css (gedit 'filename'.css)
  5. Edi the file and save it
  6. Run the python on Terminal and open the website (localhost:8000)
  7. Validate it (https://validator.w3.org/)
  8. Go back and fix the errors, starting from the top rows
  9. In Terminal, get into Mr.Keiffer's computer by running the sftp (stfp student1234@cmis-hsteacher)
  10. Get into public html folder (cd public_html) and put the file into it. (put 'filename')
